# Airship Blog

## Local dev setup

Be sure to have ruby (2.6.n) installed. RVM is
recommended to manage ruby.

This application uses Rails 6.

### Initial setup

* rvm install ruby-2.6.3 (or your ruby version manager of choice)
* bundle install
* rake db:create
* rake db:migrate
* rake db:seed
* rails s

## other notes
* rspec is the testing framework
